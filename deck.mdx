## How to check is something NaN
```
isNaN(NaN);                // true;
isNaN('NaN');              // true;
isNaN('i am not a NaN');   // true;

const customIsNaN = (instance) => instance !== instance;

customIsNaN(NaN);                // true;
customIsNaN('NaN');              // false;
customIsNaN('i am not a NaN');   // false;
